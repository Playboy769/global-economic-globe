問題核心
今年 CapEx 增加約 $30億、明年「大幅增加」，這對 foundry 客戶意味著什麼？是否代表已經拿到 Intel 14A 或 18A-P 的硬訂單？封裝與前端製造要怎麼拆分看待？
管理層回應
Dave：CapEx 相當廣泛，包含先進封裝（EMIB-T 前景令人興奮），但前端晶圓廠遠比封裝廠昂貴，因此資金仍偏向前端；投資增加是「對各業務單位客戶信心的訊號」，尤其是已簽長期協議、能據此預測未來需求的地方；重申會維持紀律，只有在確信能拿到好報酬時才投；這些晶圓廠初期是淨現金流出，這是明年 CapEx 上升的原因，Intel 7 的長壽命已證明報酬顯著。
⚑ 邏輯缺口
Ben 直接問「是否已拿到硬訂單」，Dave 完全沒有正面回答是或否，只給了「投資是信心訊號」這種定性包裝，把量化提問轉化成敘事語言——這是全場「CapEx 主旋律」的第一個範例，後面至少還會重複四次同樣的迴避形狀。
追問：如何看待競爭對手喊出 2030年 $2,200億 CPU TAM、CAGR 約45%
Dave：不願意給出具體數字，但認同市場會顯著成長；CPU 對 GPU 的比例正在接近平衡，甚至未來可能在單位數上更偏向 CPU；認為 Intel 有能力拿下顯著份額。
⚑ 邏輯缺口
被直接問到是否認同一個具體數字（$2,200億／45% CAGR），Dave 用「這是個很大的市場」帶過，既不背書也不反駁，讓分析師無法用管理層自己的口徑去校準模型。

追問：CapEx 淨額 vs 毛額，內部與 foundry 外部客戶如何拆分？
Dave：目前 gross-to-net 主要來自 AMIC／投資稅收抵免（ITC），現階段是「低個位數十億」規模，未來會更「chunky」；每一塊美國境內投資可拿回 $0.35 稅額抵免，但抵免申領有時間落差（先蓋廠→產能開始運作→設備量產就緒→才能申請→再送 IRS 審核）；投資邏輯不是先分內部/外部，而是先決定每個製程節點要多少 wafer start，再依此投資；封裝側已經看到顯著 backlog，需要提前投入內部產能並向供應商預付以取得基板供應。

問題核心
Data center 強勁大家有預期，但 client 的強勁令人意外——這是純粹靠漲價驅動的營收，還是有別的因素？相對於下半年聽起來會比往常更差的終端市場，Q3、Q4 對 client 的展望為何？
管理層回應
Dave：client 確實超乎預期，主要是 ASP 驅動（部分來自產品組合，部分是主動反映成本通膨的同類產品漲價）；年比來看市場其實是下滑的（2025年因 Windows 換機潮基期較高）；記憶體成本與供貨吃緊也讓市場走軟；Intel 把組合skew向高階，帶動 ASP；下季 client 展望「flattish」，CCPG 整體因 edge 成長會「up a bit」；市場本身偏軟、受記憶體動態拖累；因為 CPU 庫存本來就偏緊，Q3 可能會有一些 CPU 庫存回補，Q4 才會真正感受到市場走軟；好消息是這些 CPU 產能剛好可以轉去支援 data center 端無法滿足的需求。
⚑ 觀察
Dave 的回答其實相當坦誠、少見地承認「client 市場本身在走軟」，但把它包裝成對 Intel 的利多（產能可轉給 data center）——這模糊了一個問題：如果 client 真實需求在下滑，本季的 ASP 驅動成長有多少是可持續的定價權，多少只是通膨轉嫁的一次性效果？
追問：存貨減記的規模與性質？毛利率 guide flat 是否代表剔除減記後其實季減？
Dave（因通話品質問題重複了兩次才聽清楚）：部分產品在 match set（配套組合）角度沒有完全 ready，經濟上更划算的做法是轉向其他產品、不再完成這些產品，於是把這批「擱淺庫存」減記；Q3 毛利率 guide flat 部分是因為不預期重複這筆減記帶來的提升，但 Panther Lake 與 Granite（Rapids）雖然單位成本季比在改善，佔比擴大後仍低於公司平均、因為還在產品生命週期早期，這會抵銷減記帶來的提振；今年最高優先目標是讓每一季毛利率「穩穩站上 40%多」，這個目標前兩季已達成，Q3 展望顯示會延續。

Stacy 直接問「拿掉減記，毛利率 guide 是否其實隱含季減」，Dave 沒有給出剔除後的具體數字，只給了兩個抵銷因子的定性描述（減記提振 vs Panther Lake/Granite 早期低於均值的拖累）——兩個方向都沒有量化，讀者無法自行驗算 Stacy 提問背後想確認的那個數字。

問題核心
Dave 提到本季末有很多產能要上線，這是否意味著 Q4 營收會有大幅跳升？如果上季缺貨超過 $10億、9月缺貨幅度類似，Q4 是不是理應大幅成長？
管理層回應
Dave：僅guide一季是標準答案；但若 Q3末、Q4 存貨/供給改善，確實會帶來提振；即便如此，Q4 仍會「追不上」需求；內部晶圓仍吃緊，團隊執行超出預期但差距依然顯著；供給是內部晶圓＋先進封裝／基板（T-glass）＋記憶體的組合，這些恰恰是供應鏈最吃緊的環節；前端晶圓改善較線性，其他環節（封裝/基板）較「chunky」；預期 Q3 末開始看到瓶頸鬆動，這是本季「flattish」、Q4「有upside」的原因。
⚑ 邏輯缺口
Tim 的核心問題是「用同樣的缺貨幅度換算，Q4 該不該大幅跳升」，Dave 用「只guide一季」＋定性描述迴避了具體換算，「不會追上但會改善」是一個無法被驗證的模糊區間，缺貨的美元金額本身也沒有被重新確認或更新。
追問：40–60% 毛利率增量轉換率框架是否還適用於明年？
Dave：長期來看預期仍落在 40–60% 這個轉換率區間；每季有各自的動態，會落在區間低端、高端或中間，這仍是一個「相對好用的經驗法則」。
⚑ 觀察
這題管理層給出的答案本身沒有明顯迴避（區間本來就是區間），但也沒有給出 2027 落在區間哪個位置的線索——考量到 CapEx 大幅上修、Panther Lake/Granite 早期低毛利的拖累已在 Stacy 的提問中被揭露，這個區間的「中值」本身可能正在承壓，管理層沒有主動更新這點。

更值得注意的是問答環節完全沒有出現的問題：本季 GAAP 淨損 $110億、EPS $(2.16)，主要驅動來自一筆 $125億的「Escrowed Shares 按市值計價損失」——這是與美國政府 CHIPS法案 Secure Enclave 協議相關的認股結構性虧損，規模遠大於任何一題被問到的數字，卻沒有任何分析師觸及。詳見「政府持股結構深度拆解」tab 的完整分析。
本季特殊事件：CapEx 大幅上修 + 沒人問的 $125億「Escrowed Shares」按市值計價損失——CHIPS法案政府持股結構本季全面顯現的財務衝擊

本季有兩件事同時發生，管理層在問答中只讓分析師談了第一件：(1) FY2026 CapEx guidance 上修至 >$200億（設備支出年增40%），FY2027「大幅高於」2026水準；(2) GAAP EPS 從非GAAP的 $0.42 反轉成 $(2.16) 虧損，主要驅動是一筆 $125.29億的「Escrowed Shares 按市值計價損失」——與美國政府因 CHIPS法案 Secure Enclave 協議取得的認股結構有關。這筆損失比本季任何一位分析師問到的數字都大上一個量級，卻完全沒有出現在問答環節。更巧合的是，同一季的調整後自由現金流轉為 $(84.19)億，主因不是核心現金流疲弱（GAAP營運現金流仍達 $70.06億），而是一筆單季 $122.16億的「合夥人淨貢獻」流出——這很可能與 Intel 的半導體共同投資夥伴結構（如 Arizona 廠區的外部夥伴協議）有關，但截至本報告完成時尚無法從已申報文件確認具體對象與條款。

規模

  • FY2026 CapEx guidance 上修至「$200億以上」，較年初預期「顯著」提高；2026設備（tooling）支出年增40%。
  • FY2027 CapEx guidance「顯著高於」2026水準，具體數字要等一兩季後才會定案；2021–2026累計美國境內工具＋空間支出逼近$1,000億，高於任何同業。
  • Escrowed Shares 按市值計價損失：單季 $125.29億（GAAP，非GAAP調整加回項），相當於每股 $2.45——是本季GAAP轉Non-GAAP EPS橋接表中最大的單一項目，遠超股份薪酬（$0.13）與重組費用（$0.03）之和。
  • 合夥人淨貢獻/distribution：單季淨流出 $122.16億（調整後自由現金流的計算基礎），6個月累計合夥人貢獻 $40.82億、distribution $143.39億。

期限

  • Escrowed Shares 依 CHIPS Act Secure Enclave 協議，隨 Intel 履約並收到現金對價逐步釋出予美國商務部（DOC）；本季按市值計價損失反映「本季已釋出的escrow股份」與「仍留在escrow中的股份」兩者公允價值變動的合計淨額。
  • 18A-P：risk production已啟動，年底前就緒；14A：PDK 0.9 於10月到位，2027下半年risk production，2028正式量產承諾。
  • EMIB-T：backlog持續成長，2027進入客戶量產爬坡。

財務承諾／機制

  • 投資稅收抵免（ITC/AMIC）：目前抵免規模「低個位數十億」、每投入$1美元境內資本可拿回$0.35，但申領有時間落差（廠房完工→設備量產就緒→申請→IRS審核）。
  • 資產負債表：期末現金＋短期投資約$298億（含約$30B+現金頭寸管理層口徑）、$100億循環信用額度，合計約$400億流動性；另有約$100億「非核心資產」可視需要變現。
  • Q2單季發行$130億定期債（淨額）、償還$90億，長期債務餘額自年初$440.86億增至$485.49億。

核心疑問——管理層留白的盡職調查式追問清單

  2. $122.16億的單季合夥人淨貢獻流出對象是誰？是否與Arizona或其他州別的半導體共同投資夥伴（SCIP型結構）有關？若是，這類結構未來是否會持續在特定季度製造巨大的現金流波動，讓「調整後自由現金流」這個指標在CapEx上修的年度失去參考意義？

第二層（非核心會計干擾）：GAAP EPS $(2.16)、股東權益季減$232億，幾乎全數可歸因於Escrowed Shares按市值計價損失這項非現金、與CHIPS Act政府持股結構相關的項目——這不應被解讀為業務惡化，但也提醒投資人：只要Intel股價持續上漲，這類按市值計價的政府相關負債很可能在未來幾季持續產生類似的GAAP損失噪音，財報頭條數字的可讀性會被結構性拖累。
